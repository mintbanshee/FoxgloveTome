# 🌸 The Foxglove Tome

A digital sanctuary for reflection, growth, and emotional awareness.

The Foxglove Tome is a full-stack journaling application designed to provide a calm, supportive space for users to reflect on their thoughts, track their moods, and visualize their emotional journey over time.

Built with intention, this project prioritizes clarity, accessibility, and emotional comfort especially for users who may not be highly technical.

## ✨ Features
-  🌿 Journal Entries
    - Create, edit, remove and revisit personal reflections in a safe, structured space.
- 🌸 Mood Tracking
  - Select from thoughtfully organized mood categories to express how you feel.
- 🌼 Garden Visualization
  - Each week of entries blooms into a flower, creating a visual representation of emotional patterns.
  - Each month the garden resets to keep the visual current.
  - Save your garden as a png image at the click of a button.
- 🌙 Weekly Mood Summary
  - Receive a summary of your most common mood, paired with reflective quotes at the top of your dashboard.
- 🫶 Support Resources
  - Access region-based mental health resources through a simple, user-friendly interface.
- 🛡️ Secure Authentication
  - Password hashing, session-based authentication, and protected routes ensure only authorized users can access the application.
- 🔒 Privacy-Focused Access
  - Journal entries are scoped to each user, ensuring reflections remain private and accessible only to their owner.
- 🏛️ Admin Dashboard Foundation
  - Provides protected admin access, user management, mood insights, and early content management areas planned for future expansion.

## 🌿 Design Philosophy

The Foxglove Tome was created with a focus on:

- Gentle UX — calm, non-overwhelming interface
- Clarity over complexity — especially for non-technical users
- Emotional awareness — supporting reflection without judgment
- Consistency — both visually and structurally across the application

This project reflects a belief that technology can feel supportive, not stressful.

## 🌱 Stretch Goals

Future plans for The Foxglove Tome include:

- Add an account management page for users to edit their username and account details
- Complete the admin management features for users, moods, and quotes
- Add optional journal prompts/ideas through a button on the new entry page
- Build a seed shop to encourage users to grow real plants, as gardening can support mental wellness
  - Include an option for seeds curated to users' mood patterns, which is why admins can view user gardens
- Grow the garden from weekly potted plants into a full garden with a different flower for every mood option
- Add more support resources by country/region
- Add the ability to download journal entries for printing, so users can bring them to appointments or support services if needed
- Add search, sort, and filter options to the user dashboard

## 🚀 Getting Started
### PHP Version
- Place foxglove-tome-php in your local server (e.g., XAMPP htdocs)
- - Import the database
- Update database credentials if needed
- Navigate to the project in your browser
- React Version
- Navigate to foxglove-tome-react

- Install dependencies:
    - npm install
- Run the development server:
    - npm run dev
